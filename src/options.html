newBroom Settings

Configure which data to clear and which websites to protect or target.

Master Control

When enabled, newBroom will clear data based on your rules when the last browser window is closed. All changes are saved automatically.

Whitelisted Domains

Data from these domains will NOT be cleared. Use *.example.com for wildcards.

Blacklisted Domains

Data from these domains will ALWAYS be cleared. The blacklist overrides the whitelist. Use wildcards similarly.

Data to Clear on Close

Check all types of data to remove. Note: Wildcard rules only fully apply to cookies due to Chrome API limitations.
